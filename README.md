## üöÄ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- [Node.js](https://nodejs.org/) (v14 or higher) for backend sections
- [Git](https://git-scm.com/) for version control
- A code editor (VS Code recommended)
- [MongoDB](https://www.mongodb.com/) for database sections

### Installation & Setup

1. **Clone this repository:**
   ```bash
   git clone https://github.com/ggauravky/Web-Dev-Learning.git
   ```

2. **Navigate to the workspace:**
   ```bash
   cd "Ultimate Web Development Course 2025 code with harry"
   ```

3. **For frontend files (HTML/CSS/JavaScript):**
   - Open any HTML file in your browser
   - Or use Live Server extension in VS Code for hot reload

4. **For Node.js projects:**
   ```bash
   # Navigate to the Node.js folder
   cd "013 Working with Node.js"
   
   # Initialize npm (if needed)
   npm init -y
   
   # Install dependencies
   npm install
   
   # Run the server
   node index.js
   # Or use watch mode for auto-reload
   node --watch index.js
   ```

5. **For Express.js applications:**
   ```bash
   cd "014 Express.js"
   npm install express
   npm install ejs
   node index.js
   ```

6. **For MongoDB projects:**
   ```bash
   cd "015 MongoDB"
   npm install
   # Make sure MongoDB is running locally or configure connection string
   node index.js
   ```

7. **For React applications:**
   ```bash
   cd "017 React/003 Learning React"
   npm install
   npm run dev
   ```
